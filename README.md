### ✨ Featured / 精选 / 注目

| Tool | | What it does |
|------|--|--------------|
| [**botblame**](https://github.com/JaydenCJ/botblame) | `go` | Reports how much of a repo AI wrote, from commit trailers and line-level attribution. |
| [**tokdu**](https://github.com/JaydenCJ/tokdu) | `rust` | `du(1)` for tokens — recursive token-count profiler with tree view and budgets. |
| [**litegraft**](https://github.com/JaydenCJ/litegraft) | `rust` | Instant branch, snapshot and diff for SQLite files — page-level, content-addressed. |
| [**mcptap**](https://github.com/JaydenCJ/mcptap) | `go` | Transparent proxy that pretty-prints every JSON-RPC frame between an MCP client and server. |
| [**corsdoctor**](https://github.com/JaydenCJ/corsdoctor) | `go` | Explain exactly why a CORS request fails, from a captured request and response. |
| [**whydied**](https://github.com/JaydenCJ/whydied) | `go` | Explain why a process died: exit codes, signals, OOM kills, cgroup evidence. |
| [**repotrap**](https://github.com/JaydenCJ/repotrap) | `go` | Scan a freshly cloned repo for auto-execution booby traps before you open it. |
| [**harmock**](https://github.com/JaydenCJ/harmock) | `go` | Serve any HAR capture as a deterministic local mock API server. |
| [**peelback**](https://github.com/JaydenCJ/peelback) | `python` | Recursively peel base64, hex, gzip, URL and JWT layers off an opaque token. |
| [**judgecal**](https://github.com/JaydenCJ/judgecal) | `python` | Audit LLM-as-judge logs: human agreement, position bias, length bias, self-preference. |
| [**spanseq**](https://github.com/JaydenCJ/spanseq) | `typescript` | Turn an OTLP trace into a Mermaid sequence diagram or service graph. |
| [**charwash**](https://github.com/JaydenCJ/charwash) | `rust` | Detect and strip Unicode smuggling in LLM I/O: zero-width, bidi overrides, homoglyphs. |
